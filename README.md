# meeting-snap

## Agent Quickstart
- [Operations guide](docs/OPERATIONS.md)
- [Development guide](docs/DEVELOPMENT.md)

### Install
```bash
pip install -r requirements.txt
# for tests
pip install -r requirements-dev.txt
```

### Run the app
```bash
python -m t008_meeting_snap.app
```

### Run the tests
```bash
python -m pytest
```
